## üíª Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/shivsharankumar/portfolio.git
   cd portfolio
   ```

2. **Install Dependencies**
   ```bash
   npm install
   ```

3. **Start Development Server**
   ```bash
   npm run dev
   ```

4. **Build for Production**
   ```bash
   npm run build
   ```
